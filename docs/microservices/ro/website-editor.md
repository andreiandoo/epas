# Editor Vizual Website

## Prezentare Scurtă

Construiește website-uri de evenimente uimitoare fără să scrii o singură linie de cod. Editorul Vizual de Website pune designul web profesional la îndemâna ta cu o interfață intuitivă drag-and-drop pe care oricine o poate stăpâni.

Începe cu brandul tău. Panoul de personalizare a temei îți permite să definești identitatea vizuală - culori primare și secundare, tipografie din Google Fonts, layout-uri pentru header și footer. Comută suportul pentru mod întunecat cu un click. Totul se actualizează în timp real pe măsură ce designezi.

Construiește pagini bloc cu bloc. Alege din peste 25 de tipuri de blocuri concepute special pentru website-uri de evenimente. Adaugă secțiuni hero cu fundaluri uimitoare, grile de evenimente care se populează automat cu listările tale, cronometre care construiesc anticiparea și galerii de imagini care îți prezintă locațiile.

Fiecare bloc este personalizabil. Ajustează spațierea, culorile și conținutul. Adaugă butoane call-to-action care conduc vânzările de bilete. Încorporează videoclipuri de pe YouTube sau Vimeo. Creează secțiuni FAQ, testimoniale și afișaje cu logo-uri parteneri.

Constructorul de pagini suportă limbi multiple. Creează conținut în engleză și română (sau orice alte limbi) și lasă vizitatorii să aleagă preferința lor. Perfect pentru evenimente cu audiențe internaționale.

Vezi schimbările instant cu previzualizarea live. Ce vezi este exact ce vor vedea vizitatorii tăi. Fără surprize când publici.

Achiziția unică îți oferă libertate creativă nelimitată. Construiește câte pagini ai nevoie. Actualizează-ți designul oricând te inspiră. Evenimentele tale merită un website care să le egaleze excelența.

## Funcționalități

### Personalizare Temă
- Editor vizual temă cu previzualizare live
- Scheme de culori personalizate cu culori primare/secundare
- Personalizare tipografie cu Google Fonts
- Opțiuni layout header (implicit, centrat, minimal)
- Personalizare footer cu linkuri sociale
- Suport mod întunecat
- Suprascrieri CSS personalizate

### Constructor Pagini
- Constructor pagini drag-and-drop
- Peste 25 tipuri de blocuri pentru layout-uri flexibile
- Previzualizare în timp real în iframe
- Suport conținut multi-limbă (EN/RO)

### Blocuri Disponibile
- Secțiuni hero cu fundaluri și CTA-uri
- Slidere conținut cu autoplay
- Grilă evenimente cu filtrare
- Vizualizare listă evenimente
- Evidențiere eveniment principal
- Cronometru pentru evenimente
- Editor text îmbogățit
- Imagine cu legendă și lightbox
- Încorporare video (YouTube/Vimeo)
- Galerie imagini cu layout masonry
- Contor statistici cu animații
- Acordeon FAQ
- Hărți interactive
- Navigare categorii
- Butoane personalizate cu stiluri
- Linkuri social media
- Bannere CTA
- Formulare abonare newsletter
- Bannere alertă/promoție
- Carusel testimoniale
- Grilă logo-uri parteneri
- Blocuri HTML/CSS personalizate

### Tehnic
- Output prietenos SEO
- Integrare API pentru conținut dinamic
- Preseturi temă (implicit, modern, minimal, bold)

## Documentație Tehnică

### Configurare

```php
'website_editor' => [
    'blocks_count' => 25,
    'theme_presets' => ['default', 'modern', 'minimal', 'bold'],
    'supported_languages' => ['en', 'ro'],
    'google_fonts' => true,
]
```
