# Website Visual Editor

## Short Presentation

Build stunning event websites without writing a single line of code. The Website Visual Editor puts professional web design at your fingertips with an intuitive drag-and-drop interface that anyone can master.

Start with your brand. The theme customization panel lets you define your visual identity - primary and secondary colors, typography from Google Fonts, header and footer layouts. Toggle dark mode support with a click. Everything updates in real-time as you design.

Build pages block by block. Choose from over 25 block types designed specifically for event websites. Drop in hero sections with stunning backgrounds, event grids that automatically populate with your listings, countdown timers that build anticipation, and image galleries that showcase your venues.

Every block is customizable. Adjust spacing, colors, and content. Add call-to-action buttons that drive ticket sales. Embed videos from YouTube or Vimeo. Create FAQ sections, testimonials, and partner logo displays.

The page builder supports multiple languages. Create content in English and Romanian (or any other languages) and let visitors choose their preference. Perfect for events with international audiences.

See changes instantly with the live preview. What you see is exactly what your visitors will see. No surprises when you publish.

One-time purchase gives you unlimited creative freedom. Build as many pages as you need. Update your design whenever inspiration strikes. Your events deserve a website that matches their excellence.

## Technical Documentation

### Configuration

```php
'website_editor' => [
    'blocks_count' => 25,
    'theme_presets' => ['default', 'modern', 'minimal', 'bold'],
    'supported_languages' => ['en', 'ro'],
    'google_fonts' => true,
]
```
