## 3. Information We Collect

### 3.1 Account Registration Information

When you create a Tenant account, we collect:

- **Contact Information:** First name, last name, email address, phone number, position/role
- **Company Information:** Company name, Tax Identification Number (CUI), Trade Register number (Reg. Com.), VAT payer status
- **Address Information:** Street address, city, state/county, postal code, country
- **Website:** Your company or event website URL

### 3.2 Billing and Payment Information

To process payments and payouts, we collect:

- **Banking Details:** Bank name, bank account number (IBAN), SWIFT/BIC code
- **Billing Information:** Billing address, billing cycle preferences, currency preference
- **Payment Processor Credentials:** API keys for Stripe, PayU, Netopia, Euplatesc (stored encrypted)

### 3.3 Contract and Legal Information

When you sign our service agreement, we collect:

- **Contract Data:** Contract number, signing date, renewal date
- **Digital Signature Data:** Signature image/data, IP address at signing, timestamp
- **Contract Documents:** Signed contract files, contract templates used

### 3.4 Platform Configuration Data

As you configure your account, we collect:

- **Event Settings:** Event details, ticket types, pricing configurations
- **Integration Credentials:** API keys and tokens for third-party services you connect (encrypted)
- **Email Configuration:** SMTP credentials for sending emails on your behalf (encrypted)
- **Customization Preferences:** Branding, themes, templates, notification settings

### 3.5 Automatically Collected Information

When you access our platform, we automatically collect:

- **Device Information:** Device type, operating system, browser type and version
- **Log Data:** IP address, access times, pages viewed, actions taken
- **Session Information:** Session duration, features used, navigation patterns

### 3.6 Third-Party Integration Data

When you connect third-party services, we may receive:

- **Payment Processors:** Transaction status, payout information, dispute data
- **Ad Platforms:** Campaign performance data, audience insights (Google Ads, Meta, TikTok, LinkedIn)
- **CRM Systems:** Contact synchronization data (HubSpot, Salesforce)
- **Communication Platforms:** Message delivery status (WhatsApp, Twilio, Slack)

## 9. Data Security

We implement comprehensive security measures:

### 9.1 Technical Measures

- **Encryption at Rest:** Sensitive data (API keys, passwords, credentials) encrypted using AES-256
- **Encryption in Transit:** All data transmitted via TLS 1.2 or higher
- **Password Security:** Passwords hashed using bcrypt
- **Access Controls:** Role-based access control (RBAC)
- **API Security:** Rate limiting, API key authentication, HMAC signature verification

### 9.2 Organizational Measures

- Regular security audits and vulnerability assessments
- Employee training on data protection
- Incident response procedures
- Data access logging and monitoring

### 9.3 Payment Security

- PCI DSS compliant payment processing
- No storage of full credit card numbers
- Tokenization of payment methods via Stripe
